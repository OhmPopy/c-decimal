# c-decimal

C言語による10進数型の実装です。

IEEE754と異なり精度以上の数値になった場合はエラーとなります。

有効桁数、最大値、最小値等は下記のようになります。

<dl>
<dt>有効桁数</dt>
<dd>36桁</dd>
<dt>表現範囲</dt>
<dd>1.0x10<sup>-36</sup>～999999999999999999999999999999999999x10<sup>0</sup></dd>
<dt>最大値</dt>
<dd>999999999999999999999999999999999999</dd>
<dt>最小値</dt>
<dd>-999999999999999999999999999999999999</dd>
</dl>
